# Insight Buddhist Website

## Quick Start

To run this website on your local machine for development or review:

### 1. Prerequisites
Ensure you have **Python 3** installed on your system.

### 2. Install Dependencies
Install the MkDocs static site generator and the Material theme:
1. **Clone the repo:**
   `git clone https://github.com/your-username/insight-buddhist.git`
   `cd insight-buddhist`

2. **Install Dependencies:**
   `pip install mkdocs-material`

3. **Run the Site:**
   `mkdocs serve`
   Visit `http://127.0.0.1:8000` in your browser.



## Web Server Deployment

You only need a web server (Nginx, Apache, Caddy, etc.) to serve the HTML files.

1. **Generate the Build**

On your local machine or CI/CD runner, execute:
`mkdocs build`
This creates a folder named site/. This folder contains the entire website.

2. **Upload to Server**
Upload the contents of the site/ folder to your web server's root directory (e.g., /var/www/html/ or public_html/).

3. **Server Configuration (Nginx Example)**

To ensure clean URLs and proper handling of 404s, use a configuration similar to this:
---
   server {
       listen 80;
       server_name insightbuddhist.org;
       root /var/www/html;
       index index.html;

       location / {
           try_files $uri $uri/ =404;
       }

       # Custom 404 page
       error_page 404 /404.html;
   }
---
## Directory Structure

docs/: The core content folder. All English pages are stored here as .md files.

docs/images/: The centralized media library.

mkdocs.yml: The "brain" of the site. Controls navigation, theme colors, and plugins.

site/: DO NOT EDIT. This is the output folder generated by the build command.

### NOTES

Portability: The entire site can be moved to a new host simply by moving the site/ folder.
Editing: Modify the .md files in the docs/ directory.
Navigation: Update the nav section in mkdocs.yml to change the menu structure.
Images: Reference images using relative paths: `![Alt Text](images/filename.jpg)`
